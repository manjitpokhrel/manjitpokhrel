## Research

**NASB — Nepali Adversarial Safety Benchmark**
Structured multilingual adversarial benchmark evaluating tokenizer fragmentation and alignment robustness across Qwen, Gemma, Llama, and Gemini. 1,200+ probes across multilingual and code-switched attack surfaces. 73.7% bypass rate in Nepali. 0% in English.

**Vajra Morphing**
Novel sub-tokenization attack class exploiting Devanagari/Latin code-switching vulnerabilities to fragment safety-aligned tokenizer boundaries.

**GhostWeight — Training-Free LLM Inference Optimization**
Sparse inference framework achieving up to 110.5% hardware speedup using activation sparsity and custom CUDA kernels on RTX 5060 hardware. No fine-tuning. No retraining.

**LLM Red Teaming & Alignment Analysis**
Systematic prompt injection, jailbreak evaluation, and multilingual safety testing across open-weight LLM ecosystems.

**MiniGPT — Transformer From Scratch**
211K-parameter GPT-style transformer implemented entirely in NumPy with manual backpropagation and attention mechanisms.
